Requirements:
In order to run the _.py files, Python 2.7 must be installed.
In order to run the _.ipynb files, Anaconda is recommended to use.

RunCode:

2.split_data.py
3.smaller_data.py
4.validating_data.py
5.Project5_Predict_future_success.ipynb
